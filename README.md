## 🚀 Main part 
* **Місцерозташування:** Інтерактивна карта (перша лінія, вул. Приморська, 13).
* **Меню ресторану:** Список страв європейської та одеської кухні (гриль-меню, морепродукти, комплексні обіди).
* **Інфраструктура:** Опис басейну, зони барбекю та прямого виходу до моря.

## 🛠 Tech
* **Frontend:** HTML5, Tailwind CSS, JavaScript.
* **Maps:** Google Maps / iframe.
* **Deployment:** GitHub Pages.

## 📍 Контакти готелю
* **Адреса:** Україна, Одеська обл., с. Грибівка, вул. Приморська, 13.
