## Особливості проєкту

* **Мобільна адаптивність (Mobile-First):** Повноцінне виїзне бічне меню (Off-canvas) для смартфонів з плавними CSS-анімаціями та `backdrop-blur` оверлеєм.
* **Сучасна стилізація:** Використання новітнього фреймворку **Tailwind CSS v4** з кастомними налаштуваннями теми (шрифти, кольори) через `@theme`.
* **Продуктивність (Zero Dependencies):** Логіка інтерфейсу (модальні вікна, скролл, відкриття меню) написана на чистому **Vanilla JavaScript** без використання важких бібліотек типу jQuery.
* **Типографіка та UI:** Інтегровано преміальний шрифт *Manrope* (Google Fonts), налаштовано антиаліасинг, використані легковагові inline SVG-іконки.
* **Hero Section:** Динамічний перший екран з фоновим зображенням, градієнтним затемненням та мікроанімаціями кнопок при наведенні.

## Технологічний стек

* **Кістяк:** HTML5, Semantic Markup
* **Стилі:** Tailwind CSS (v4)
* **Логіка:** JavaScript (ES6+)
* **Збірка та розробка:** Vite 
* **Деплой:** GitHub Pages
